# ADR-001 Desktop Time Owns Growth

## Decision

Desktop Time owns Core Growth.

## Reason

Core Growth must remain tied to actual desktop companionship time.

## Alternatives Considered

- Offline Growth owns Core Growth.
- Behavior owns Core Growth.
- Reveal owns Core Growth.

## Impact

Core Growth has one primary owner.

Other systems may read Core Growth or modify efficiency only through defined responsibilities.

## Status

Frozen
