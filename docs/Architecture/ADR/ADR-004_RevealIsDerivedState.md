# ADR-004 Reveal Is Derived State

## Decision

Reveal is Derived State.

## Reason

Reveal must be reconstructable from Persistent State and architecture-defined calculation.

## Alternatives Considered

- Save Reveal as authoritative state.
- Let CatBrain own Reveal.
- Let Behavior own Reveal.

## Impact

Reveal can be recalculated.

Reveal does not own Core Growth.

## Status

Frozen
