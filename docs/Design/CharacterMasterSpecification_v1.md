# Character Master Specification v1.0

## Status

Yanlai v1.0 Character Master is frozen.

This document is the highest visual authority for Yanlai character production art. The three Character Master Reference images in `assets/character_master/` are the only active Character Master references:

- `assets/character_master/CMR_001_Resting.png`
- `assets/character_master/CMR_002_Calm.png`
- `assets/character_master/CMR_003_Sitting.png`

Previous concept art, exploratory images, abandoned references, and outdated cat reference images are historical only. They are archived under `assets/archive/concept_references/` and must not be treated as active references.

## Character Master Root States

Yanlai v1.0 has three semantic Character Master root states. Future animation and sprite production must choose the correct CMR root before creating action frames.

### CMR_001_Resting.png - Rest Root

Rest Root is the default self-state.

Used for:

- Idle
- Blink
- Ear micro movement
- Tail micro movement
- Normal breathing
- Default low-interruption desktop presence

Meaning:

Yanlai is awake, relaxed, and simply existing beside the user.

### CMR_002_Calm.png - Relax Root

Relax Root is the deeper relaxation self-state.

Used for:

- Deep relax
- Yawn
- Sleep transition
- Sleep
- Long resting state
- Low posture states

Meaning:

Yanlai has lowered its body and entered a more relaxed, sleepy, or long-resting state.

### CMR_003_Sitting.png - Interaction Root

Interaction Root is the external-response state.

Used for:

- Petting response
- Mouse attention
- Head tilt
- Observe cursor
- Watch toy
- Watch butterfly
- Interact with furniture or objects
- Curious but non-alert response

Meaning:

Yanlai has noticed or is responding to something outside itself.

It is more expressive than CMR_001_Resting.png and CMR_002_Calm.png, but must remain calm, non-performative, and low-interruption.

### Root State Principle

Yanlai expresses companionship through states, not dramatic emotions.

CMR_001_Resting.png and CMR_002_Calm.png represent self-states.

CMR_003_Sitting.png represents interaction-state.

## Production Inheritance Rule

All future production art must inherit from the three frozen Character Master Reference images.

This applies to:

- Idle
- Walk
- Sit
- Sleep
- Pet
- Reveal
- Animation
- Sprite Sheets
- Promotional artwork

Character proportions and visual language may not be changed unless a future Character Master v2.0 is explicitly created.

## Frozen Character Language

The permanent Yanlai v1.0 character language is:

- Rounded horizontal-oval face
- Compact facial features
- Small nose
- Very small mouth
- Facial features grouped toward the center
- Full cheeks
- Wide and low body
- Low center of gravity
- Short legs
- Compact rounded body
- Relaxed posture
- Calm presence
- Companion before performance
- No exaggerated emotional expression
- One-meter readability

## Companion Philosophy

Yanlai is not performing.

Yanlai is not asking for attention.

Yanlai is not expressing strong emotions.

Yanlai exists quietly beside the user.

Yanlai communicates through comfortable states rather than dramatic emotions.

The preferred emotional direction is:

- Calm
- Warm
- Comfortable
- Relaxed
- Content

The following emotional directions are not part of Yanlai:

- Sad
- Melancholic
- Pitiful
- Anxious
- Overly excited
- Attention-seeking
- Overly anthropomorphic facial acting

## Core Character Design Philosophy

The goal is not maximum realism.

The goal is not maximum illustration quality.

The goal is not exaggerated cuteness.

The goal is a companion users instinctively want to gently pet.

Yanlai should feel like it has peacefully lived beside the user for a long time.

Soft Character Design is prioritized over rendering complexity.

Large readable silhouettes are more important than fine details.

Production assets must always be evaluated at actual runtime size rather than enlarged artwork.

## Freeze Rule

Future production artwork starts from Character Master, not from generating new concept cats. Character exploration is complete for Yanlai v1.0. Future work focuses on production assets derived from the frozen Character Master.
