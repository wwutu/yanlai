# ADR-006 Single Writer Principle

## Decision

Core Growth follows the Single Writer Principle.

## Reason

Core Growth must have one authoritative writer to avoid conflicting progression sources.

## Alternatives Considered

- Multiple systems write Core Growth.
- Behavior writes Core Growth directly.
- Reveal writes Core Growth directly.

## Impact

Core Growth ownership remains clear.

Other systems must not directly write Core Growth.

## Status

Frozen
